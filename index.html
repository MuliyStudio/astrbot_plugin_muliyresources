安装方式

两种方式任选其一

① 插件市场（推荐）

打开 AstrBot WebUI → 插件管理，搜索「暮黎资源聚合」点击安装，按需填写配置即可。

# 需填写的配置 xdgame_username / xdgame_password # 游戏 Cookie 刷新 muliy_username / muliy_password # 教父.com 新站影视源（可选） wyy_custom_url # 网易云解析后端（必填才可用）

② 手动安装

克隆到 AstrBot 插件目录后重启。

cd <astrbot>/data/plugins/ git clone https://github.com/MuliyStudio/astrbot_plugin_muliyresources.git # 重启 AstrBot
